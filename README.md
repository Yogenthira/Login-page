# Login-page
Login Page using React
Overview
This project is a simple login page built using React. It includes a stylish login form with username and password input fields, along with a login button. The design follows modern UI principles with a dark theme and supports both light and dark modes.

Features:
Responsive design for mobile and desktop

Dark and light mode support:

Styled using modern CSS techniques

Secure handling of user credentials (basic implementation)

Clean and maintainable code structure

Technologies Used:

React.js

CSS (Flexbox, Grid, Media Queries)

React Hooks (useState, useEffect)
